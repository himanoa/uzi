Data.Uzi.HeartbeatInterval

Description

Discordのハートビートリクエストを送る間隔を保存するためのデータ構造を定義したモジュールです。 ハートビートの間隔は DiscordGatewayAPIのHelloEventによって受信し、'State HeartbeatInterval' に保存します。

State HeartbeatInterval を読みこんで間隔通りにHeartbeatリクエストを送るのは sendHeartbeat です

DiscordAPI側のドキュメント: https://discord.com/developers/docs/topics/voice-connections#heartbeating |
