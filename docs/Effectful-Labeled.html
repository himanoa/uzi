Description

Labeled effects.

Synopsis
  • data Labeled (label :: k) (e :: Effect) :: Effect
  • runLabeled :: forall label e es a b. (Eff (e : es) a -> Eff es b) -> Eff (Labeled label e : es) a -> Eff es b
  • labeled :: forall label e es a. Labeled label e :> es => Eff (e : es) a -> Eff es a

Example

An effect can be assigned multiple labels and you can have all of them available at the same time.

>>> import Effectful.Reader.Static

>>> :{
 action
   :: ( Labeled "a" (Reader String) :> es
      , Labeled "b" (Reader String) :> es
      , Reader String :> es
      )
   => Eff es String
 action = do
   a <- labeled @"b" @(Reader String) $ do
     labeled @"a" @(Reader String) $ do
       ask
   b <- labeled @"b" @(Reader String) $ do
     ask
   pure $ a ++ b
:}

>>> :{
 runPureEff @String
   . runLabeled @"a" (runReader "a")
   . runLabeled @"b" (runReader "b")
   . runReader "c"
   $ action
:}
"ab"
