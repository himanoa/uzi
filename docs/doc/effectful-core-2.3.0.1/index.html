effectful-core-2.3.0.1: An easy to use, performant extensible effects library.

An easy to use, performant extensible effects library with seamless integration with the existing Haskell ecosystem.

This library provides core definitions with a minimal dependency footprint. See the effectful package for the "batteries-included" variant.
